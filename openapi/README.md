# Savvy Rilla FX API OpenAPI Foundation

This folder contains the OpenAPI foundation for Savvy Rilla FX API.

## Files

- `openapi.json` — static OpenAPI 3.1 document.
- `openapi.yaml` — YAML version for import into tools.
- `/openapi.json` — runtime route generated from `lib/openapi`.
- `/openapi.yaml` — runtime YAML route generated from `lib/openapi`.
